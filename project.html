In multivariable calculus, partial differentiation measures how a function changes as one variable varies, keeping all other variables fixed.

📘 Theory: Variable treated as constant in Partial Differentiation

A. Definition: Partial differentiation is the process of differentiating a function of several variables with respect to one of those variables, while holding the remaining variables constant. In other words, when computing ∂f/∂x, treat y, z,... as constants. This "treating as constant" is the core idea that simplifies multivariable derivatives.

B. Mathematical Explanation:
Consider a function of two variables: z = f(x, y). Then:

∂z/∂x = derivative with respect to x, treating y as constant.
∂z/∂y = derivative with respect to y, treating x as constant.

Example: If z = x²y + 3xy²

∂z/∂x = 2x·y + 3·(1)·y² = 2xy + 3y² → (y treated constant, differentiate x² → 2x, 3xy² → 3y²)
∂z/∂y = x²·(1) + 3x·(2y) = x² + 6xy → (x treated constant)

C. Step-by-step method:

  • Step 1: Identify the variable with respect to which you differentiate.
  • Step 2: Treat all other independent variables as constants (like numbers).
  • Step 3: Apply ordinary differentiation rules (power rule, product rule, chain rule).

D. Key Rules (same as ordinary derivatives):

  • Power Rule: ∂(xⁿ)/∂x = n·xⁿ⁻¹ (others constant).
  • Constant Rule: derivative of a constant term (w.r.t variable) = 0.
  • Product Rule: ∂(u·v)/∂x = u·(∂v/∂x) + v·(∂u/∂x) when both u and v involve x.

E. Applications in Engineering & Physics:
Partial differentiation is fundamental in modeling heat transfer (temperature gradient), fluid dynamics (velocity fields), elasticity, economics, and optimization problems where multiple parameters simultaneously influence outcomes. The concept "treat other variables as constant" allows engineers to analyze sensitivity along one dimension at a time.
